This folder contains resources and documentation associated with the 2016 RDM Librarian Training for the Sciences, Workshop.

Workshop 1 Agenda is the training outline.
Subject_Librarians.csv contains the data that librarians will update locally and push (make a "pull request") back to the GitHub repository.
Note that the Bibliometric data folder -- containing 5 xls spreadsheets with metadata generated from Web of Science citation reports -- has been moved to Workshop 2.
Download the lesson outline (Sciencelibrariantraining-gitandGitHub-lessonoutline.pdf) to access the links.
